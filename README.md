# 🔐 Network Scanning using Nmap

## 🎯 Objective
The goal of this project is to identify open ports and services on a target system using Nmap.

---

## 🛠 Tools Used
- Kali Linux 
- Nmap   

---

## ⚙️ Methodology

### 1. Basic Scan
nmap <target-ip>

### 2. Service Detection
nmap -sV <target-ip>

### 3. OS Detection
nmap -O <target-ip>

---

## 📊 Results
- Port 22 (SSH) open  
- Port 80 (HTTP) open  
- Apache server detected
